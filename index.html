2D POUND

An Easy, and effective way of adding collision particles to your game!

(GIF of 2D Pound in action. A square colliding with a surface.)

How to use

        Step 1:  Add the TwoD_PoundGround script to any object you want the particles to come       from.

        

(GIF showing how to add the TwoD_PoundGround script to a GameObject.)

Step 2:  Tweak TwoD_PoundGround settings.   Change the “Material” variable to whatever you want, but type anything too crazy, we’ll need this variable later.

        Under the settings panel, change the “Sides of object to use Pound Effect on.” variables to whatever you like (ex:         

                                Left - The pound effect will work on the left side if this is enabled.

                                Right- The pound effect will work on the right side if this is enabled.

                                And so on…)

The OnCollide() variable is a UnityEvent, they are super cool, check them out.

(My settings for my Green Square’s TwoD_PoundGround component)

Step 3: Add a TwoD_Pound script to any object you want the TwoD_PoundGround objects to collide with. (ex: The player)

                

Step 4: Add pound effects to the Pound Effects array.

        

Step 5: Set Materials of Pound Effects, The “Material” variable is to identify which pound effect to use for certain objects. For example, on my Green Square’s TwoD_PoundGround script, I set the Material to “Green Square”, so if I set the Material variable for one of my Pound effects to “Green Square” and Press play, My 2D POUND will be finished!

                Extra Info: If you have multiple Pound Effects with the same Material, they system will choose randomly from those multiple pound effects.

Step 6: You’re DONE!
